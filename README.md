To Run this project, you have to follow the following steps,
- In the editor, for testing, run the ProcGeneration Scene
<img width="445" height="36" alt="image" src="https://github.com/user-attachments/assets/4f800ce3-be2b-4e73-b4ef-8df2f8e35258" />

- In the editor, to play the game, run the LobbyScene scene,
<img width="404" height="36" alt="image" src="https://github.com/user-attachments/assets/821d7a2a-c0de-42b6-bbfc-b0a5e59c5422" />
- Press "Play Game"
- Enter your IPAddress or localHost
- Press "Create Game"
- Select your role (For singleplayer, use the survivor role)
- Press "Ready Up"
- Wait...

- In the game build, follow the editor play steps from inside the build
